1) CSS references style.css @import url('/style/person.css') @import url('/style/animal.css') Uses the @import rule which imports one or more stylesheets into HTML files or into another CSS file. This method can be used to import all the smaller CSS files into a single CSS file, which is then loaded into the HTML document between a style tag in the head of the document. This method isn't used often these days because the downside to using import is the potential for slowing down your page speed.
