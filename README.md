### Running Your First Project

#### Node.js: Express Basics
```bash
cd node-dev/0008-express-basics
npm install
npm start  # Server runs on http://localhost:3000
```

#### React: Hello React
```bash
cd react/hello-react
npm install
npm start  # Dev server on http://localhost:3000
```

#### Angular: Reddit Clone
```bash
cd angular/reddit-clone
npm install
ng serve  # Dev server on http://localhost:4200
```

#### Python: Simple Script
```bash
cd python/bootcamp
python main.py
```

#### Go: Hello World
```bash
cd golang/hello-world
go run main.go
```
